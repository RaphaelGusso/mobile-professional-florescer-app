# 🌱 Flourish -- Gerenciador de Plantas

Aplicativo desenvolvido em **React Native + Expo**, criado para ajudar
no cuidado de plantas.\
Permite cadastrar plantas, salvar fotos, armazenar tudo no Firebase e
receber lembretes de rega por notificações.

## 📱 Funcionalidades

- Cadastro de plantas
- Firebase Firestore + Storage
- Notificações locais
- NativeWind
- expo-router

## 🛠 Tecnologias Utilizadas

React Native, Expo, expo-router,
expo-notifications, Firebase, NativeWind, TypeScript.

## 🚀 Como Rodar o Projeto

```sh
npm install
npx expo start
```
